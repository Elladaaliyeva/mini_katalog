# 🛍️ Mini Katalog Uygulaması

Flutter ile geliştirilmiş basit bir ürün kataloğu uygulaması.

## Özellikler
- Ürün listesi (GridView)
- Ürün arama ve filtreleme
- Ürün detay sayfası
- Sepete ekleme

## Flutter Sürümü
Flutter 3.41.9

## Çalıştırma Adımları
1. `flutter pub get`
2. `flutter run`

## Kullanılan API
https://fakestoreapi.com/products
